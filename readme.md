# 🧙🏻‍♀️ 마법옷장
당신을 위한 **마법옷장**,    
더 이상 오늘 날씨에 어떤 옷을 입을지 고민하지 마세요!   

**마법옷장**은 날씨에 따라 당신에게 어울리는 옷을 추천해드리는 어플리케이션입니다.

본 readme는 `서버` repository의 `readme`입니다.     
**어플리케이션 기능 및 전체적인 UI는 프론트엔드 repository**에 있으니,     
이 곳의 readme를 보고 싶으신 분께서는 **아래 링크로 이동**해주세요!
